# Bank Transaction Analysis using MySQL

## Project Overview
This project analyzes bank transaction data using MySQL to identify customer spending patterns, transaction trends, and business insights.

## Tools Used
- MySQL
- MySQL Workbench
- GitHub

## Dataset
Bank transaction dataset containing customer details, transaction amount, account balance, location, gender, and transaction date.

## SQL Concepts Used
- Aggregate Functions (COUNT, SUM, AVG)
- GROUP BY
- ORDER BY
- LIMIT
- Window Functions (RANK)

## Key Insights
- Top spending customers identified
- Location-wise transaction analysis
- Gender-wise spending analysis
- Monthly transaction trends
- Customer ranking based on spending
